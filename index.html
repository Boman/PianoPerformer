PianoPerformer

This project aims to create a HTML5 code basis for piano players to read in their .mid files and let them show the keys to press in a waterfall like animation.

planned features:

  • Midi Keyboard input for interactive speed mode

libs:

  • jasmid (Matt Westcott & Ben Firshman)
